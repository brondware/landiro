# Landiro CMS

Легка PHP CMS без бази даних для створення та керування мобільними лендингами з високою конверсією. Розроблена спеціально для товарного e-commerce, дропшипінгу та прямого маркетингу.

Без фреймворків. Без бази даних. Тільки PHP 8.1+ і JSON-файли — розгортається на будь-якому хостингу за лічені хвилини.

---

## Можливості

### Конструктор лендингів
- **Візуальний редактор секцій** — перетягуйте секції, змінюйте порядок
- **18 типів секцій**: Hero/Банер, Переваги, Продукт, Як це працює, Відгуки, Ціна/Оффер, Таймер зворотного відліку, FAQ, Галерея, Відео, Форма замовлення, Довіра, CTA-кнопка, Підвал, Текстовий блок, До/Після, Кастомний HTML та інші
- **Готові шаблони секцій** — редаговані змінні (тексти, кольори, зображення, стилі)
- **Шаблони лендингів** — починайте з blank, product, service або info
- **Пресети секцій** — зберігайте і повторно використовуйте власні конфігурації
- **Історія версій** — до 10 збережених версій на лендинг, відновлення в один клік
- **A/B тестування** — розбивка трафіку 50/50 на основі cookie з аналітикою
- **Клонування лендингів** — дублювання будь-якого лендингу разом із секціями
- **Швидкий перегляд** — десктоп і мобільний preview без виходу з редактора

### Управління замовленнями (CRM)
- **Статуси замовлень** — Нове / Передзвонено / Підтверджено / Скасовано з кольоровими бейджами
- **Нотатки inline** — додавайте нотатки прямо в таблиці замовлень
- **Відстеження виручки** — ціна замовлення, загальна виручка, середній чек, кількість
- **Фільтрація по статусах** — таби з лічильниками по кожному статусу
- **Експорт у CSV** — всі замовлення зі статусами, цінами та нотатками

### Маркетингові інструменти
- **Таймер зворотного відліку** — sticky-панель зверху з сесійним або фіксованим дедлайном; налаштовуваний текст і кольори
- **Соціальний доказ** — спливаючі сповіщення про покупки («Олег з Києва щойно замовив…»), налаштовуваний delay та інтервал
- **Конструктор попапів** — попап з exit-intent або таймером, кастомний контент і CTA
- **Sticky CTA-панель** — завжди видима смуга із закликом до дії внизу сторінки
- **Плаваючий віджет** — кнопка WhatsApp, Viber, Telegram або дзвінка, зафіксована на екрані

### Аналітика
- **Підрахунок переходів** — унікальні та загальні відвідувачі на кожен лендинг
- **Відстеження замовлень** — кількість і відсоток конверсії
- **Результати A/B-тесту** — порівняння конверсій між варіантами секцій
- **KPI виручки** — загальна виручка, середній чек
- **UTM-трекінг** — передача UTM-параметрів у замовлення через sessionStorage

### Інтеграції
- **Telegram-сповіщення** — миттєві повідомлення про нові замовлення в будь-який бот або канал
- **Email-сповіщення** — відправка замовлень через PHP mail()
- **Webhook** — POST JSON-даних замовлення на будь-який ендпоінт (CRM, Zapier, Make тощо)
- **Facebook Pixel** — pixel ID на кожен лендинг з подіями PageView і Lead
- **TikTok Pixel** — pixel з подіями ViewContent та Lead
- **Snapchat Pixel** — підтримка пікселя Snapchat

### SEO та технічне
- **SEO на кожен лендинг** — title, meta description, Open Graph зображення
- **Кастомні скрипти** — вставка скриптів у `<head>` та `<body>` на рівні лендингу
- **Захист паролем** — обмеження доступу до лендингу PIN-кодом
- **Автовизначення BASE_URL** — працює в корені і підпапці на будь-якому хостингу
- **QR-код лендингу** — миттєва генерація і завантаження QR у PNG
- **Конструктор форм** — кастомні поля (текст, телефон, email, select, checkbox) з drag-to-reorder

### Медіа та контент
- **Медіа-менеджер** — завантаження, перегляд, копіювання URL і видалення зображень
- **Підтримувані формати** — JPEG, PNG, GIF, WebP, SVG (до 5 МБ)
- **Слайдер До/Після** — порівняння зображень перетягуванням або свайпом
- **Video Hero** — секція з відео на YouTube з lazy-load

### Адмін-панель
- **Мобільна адаптивність** — hamburger-меню, overlay, підтримка клавіатури
- **Імпорт / Експорт лендингів** — ZIP-архів з лендингом і медіафайлами
- **Масові операції** — публікація, приховування, видалення кількох лендингів одночасно
- **Маркетплейс шаблонів** — встановлення шаблонів секцій із ZIP-файлів
- **UTM-генератор** — конструктор UTM-посилань із копіюванням в буфер

### Безпека
- **CSRF-захист** — токени на всіх API-викликах і HTML-формах
- **Блокування брутфорсу** — 5 невдалих спроб → IP-блокування на 5 хвилин
- **Захищені cookie сесії** — `HttpOnly`, `SameSite=Strict`, `Secure` (на HTTPS)
- **Самовидалення setup.php** — файл налаштування видаляється автоматично після першого запуску
- **Захищена директорія data/** — `.htaccess` блокує прямий доступ до всіх файлів
- **Вимкнений display_errors** — помилки не відображаються у браузері

## Вимоги

- **PHP 8.1+**
- **Apache** з увімкненим `mod_rewrite` (для `.htaccess`)
- PHP-розширення: `ZipArchive`, `json`, `fileinfo`
- Директорія `data/` має бути доступна для запису веб-сервером

## Встановлення

**1. Завантажте файли**

Завантажте весь проєкт у веб-корінь сервера або підпапку через FTP, SSH або файловий менеджер хостингу.

**2. Встановіть права доступу**

```bash
chmod 755 data/
chmod 644 config.php
```

**3. Запустіть майстер налаштування**

Відкрийте `https://yoursite.com/setup.php` у браузері. Вкажіть логін і пароль адміна. Файл setup.php **видаляється автоматично** після збереження.

**4. Увійдіть в адмін-панель**

Перейдіть на `https://yoursite.com/login.php` і введіть щойно встановлені дані.

**5. Створіть перший лендинг**

В адмін-панелі натисніть **Новий лендинг**, оберіть шаблон, додайте секції та опублікуйте.

## Структура директорій

```
landiro-cms/
├── admin/              # Сторінки адмін-панелі та API
│   ├── api.php         # JSON API ендпоінт
│   ├── partials/       # Sidebar та спільні блоки
│   └── ...
├── assets/             # CSS і JS для адмінки
├── core/               # Базові класи (Auth, Landing, Renderer тощо)
├── data/               # Робочі дані — не включаються в git
│   ├── landings/       # JSON-файли лендингів
│   ├── orders/         # JSON-файли замовлень
│   ├── analytics/      # JSON-файли аналітики
│   ├── uploads/        # Завантажені медіафайли
│   └── settings.json   # Глобальні налаштування (Telegram, email, webhook)
├── landings/           # Публічний роутер лендингів
├── templates/          # HTML/CSS/JS шаблони секцій
├── config.php          # Конфігурація та константи
├── login.php           # Вхід в адмінку
└── setup.php           # Первинне налаштування (видаляється після використання)
```

## Конфігурація

Усі налаштування керуються через адмін-панель. Після встановлення можна налаштувати:

- **Telegram Bot** — токен і chat ID для сповіщень про замовлення
- **Email** — адреса отримувача листів із замовленнями
- **Webhook URL** — ендпоінт для зовнішніх інтеграцій

Розширені параметри (BASE_URL, шляхи, ліміти завантажень) знаходяться у `config.php`.
